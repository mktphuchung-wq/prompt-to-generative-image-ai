# MVP Roadmap

## Phase 1: Product Foundation

- Finalize product documentation.
- Define data model and storage choices.
- Confirm Side Panel workflow and external tool boundaries.

## Phase 2: Project and Template Basics

- Implement Side Panel shell.
- Add lightweight Projects with a maximum of 5 Projects.
- Add Prompt Template CRUD.
- Add basic node canvas editing.

## Phase 3: Prompt Graph MVP

- Connect prompt nodes.
- Compile connected nodes into a final prompt.
- Add prompt preview and manual edit support.
- Save Prompt Runs.

## Phase 4: Tool Handoff and Outputs

- Add semi-auto handoff adapters for ChatGPT, Gemini, and Google Flow.
- Allow generated image attachment as Outputs.
- Add Output review metadata.

## Phase 5: Artifact Library

- Add explicit Output approval.
- Promote approved Outputs into the Artifact Library.
- Add Artifact browsing and reuse metadata.
