# UI/UX Spec

## Primary Surface

The Side Panel is the main extension UI. It should provide persistent access while users browse and work in external AI tools.

## Main Areas

- Project switcher with a maximum of 5 Projects.
- Prompt Template list and editor entry points.
- Node canvas for composing prompt templates.
- Final prompt preview panel.
- Tool handoff controls for ChatGPT, Gemini, and Google Flow.
- Prompt Run history with attached Outputs.
- Output review area.
- Artifact Library for approved reusable assets.

## UX Principles

- Keep the MVP semi-auto and user-controlled.
- Make the distinction between temporary Outputs and approved Artifacts visible.
- Avoid Campaign Manager concepts such as campaign calendars, budgets, channels, and complex approval pipelines.
- Prefer clear workflow steps over hidden automation.
