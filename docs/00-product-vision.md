# Product Vision

AI Visual Workflow Extension is a Chrome Extension Side Panel assistant for Content Creative users who generate high volumes of AI images, product mockups, visual variations, and campaign assets.

The product helps users turn repeatable visual ideation work into structured prompt workflows without becoming a full Campaign Manager. It focuses on lightweight Projects, node-based prompt templates, semi-automated prompt execution support, temporary generated Outputs, and approved reusable Artifacts.

The MVP should make prompt construction, tool handoff, output review, and artifact approval faster while keeping users in control of external AI tools and account interactions.
