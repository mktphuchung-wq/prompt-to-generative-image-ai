# Feature Spec

## Core MVP Features

- Chrome Extension Side Panel as the primary UI.
- Lightweight Projects, limited to 5 total Projects.
- Project-scoped Prompt Templates, Prompt Runs, Outputs, Source Images, and Artifact Library.
- Node-based Prompt Template canvas.
- Prompt node connection and final prompt generation.
- Prompt preview and manual editing before use.
- Semi-automatic tool handoff for ChatGPT, Gemini, and Google Flow.
- Generated image attachment as temporary Outputs.
- Output review, metadata editing, and approval.
- Artifact Library for user-approved reusable assets.

## Explicit Non-Goals

- Do not build a full Campaign Manager.
- Do not implement full account automation.
- Do not bypass captchas.
- Do not automate behavior that violates external tool Terms of Service.
- Do not build the extension UI or app logic as part of the initial documentation task.
